# Pokedex - Android


El proyecto fue desarrollado con fines educativos.

El objetivo del proyecto es mostrar información detallada sobre los pokemones.

### Arquitectura
- Clean architecture

### Patron de diseño
- MVVM


### Tecnologias usadas
 
- Live data
- View Model
- View Binding
- NavigationComponent
- Retrofit 2
- Picasso
- com.daimajia.numberprogressbar
- Gson
- coroutines
